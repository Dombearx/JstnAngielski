# JstnAngielski

Strona do nauki języka angielskiego, hostowana na GitHub Pages.

## 🌐 Strona

Po włączeniu GitHub Pages witryna będzie dostępna pod adresem:  
`https://dombearx.github.io/JstnAngielski/`

## 🚀 Wdrożenie

Strona jest automatycznie wdrażana na GitHub Pages przy każdym wypchnięciu zmian do gałęzi `main` za pomocą GitHub Actions (`.github/workflows/deploy.yml`).

## Struktura

```
index.html   – główna strona
```
